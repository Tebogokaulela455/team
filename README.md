# Lesedi PAS Backend (Fixed)

## Setup

1. npm install
2. npm start

## Endpoints

- POST /api/auth/register
- POST /api/auth/login
- POST /api/subscription/pay
- CRUD /api/members
- CRUD /api/dependants
- CRUD /api/policies
- POST /api/payments
- POST /api/notifications/reminders
- GET /api/documentation/:id
- POST /api/claims
- CRUD /api/agents
- POST /api/retail/sync
- POST /api/lapse/run
- GET /api/reports/:type
